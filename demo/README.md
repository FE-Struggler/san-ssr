# SSR Demo

## SSR in PHP

Call `ssp -t php` to generate ssr.php:

```bash
npm run build-php
```

Run render.js to call ssr.php and render HTML:

```bash
npm run render-php
```

## SSR in JavaScript

Call `ssp -t js` to generate ssr.php:

```bash
npm run build-js
```

Run render.js to call ssr.php and render HTML:

```bash
npm run render-php
```
